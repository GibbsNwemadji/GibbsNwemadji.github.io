# Gibbs Nwemadji academic website

This is a lightweight AcademicPages-style organization with a custom visual style.

## How to edit

- Edit `_data/news.yml` to update the top news block.
- Edit `_data/events.yml` to update talks, posters, schools, workshops, and conferences.
- Edit `_data/publications.yml` to update publications.
- Edit `_data/path.yml` to update education and visits.
- Edit `_data/research.yml` to update research cards.
- Edit `assets/css/style.css` to change colors, typography, spacing, and layout.

## Profile image

The site currently expects the profile image here:

```text
assets/Gibbs_Nwemadji.png
```

If you prefer to move the image into `assets/img/`, update `profile_image` in `_config.yml`.

## Deploying on GitHub Pages

Put these files at the root of your repository `GibbsNwemadji.github.io`. GitHub Pages will build the site with Jekyll.

Go to:

Settings → Pages → Build and deployment → Deploy from a branch → main → root

Then open:

```text
https://GibbsNwemadji.github.io
```
